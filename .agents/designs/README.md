## Design Guidelines

### Reference Websites
- [aayushbharti.in](https://aayushbharti.in/) - Animation inspiration, dark mode
- [visuvate.com](https://www.visuvate.com/) - Clean agency aesthetic
- [davidlozano.pro](https://davidlozano.pro/) - Spanish market, bold typography

### Key Design Principles
1. Premium, modern feel - not a basic MVP
2. Dark and light mode support
3. Mobile-first responsive design
4. Rich micro-animations
5. Service pills color-coded by category

## How to Use

1. Before implementing any component, check this folder for reference designs
2. If a design exists, match it as closely as possible
3. If a design is missing or ambiguous, ask for clarification before improvising
4. Add new design files here when received from stakeholders
